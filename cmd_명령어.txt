현재 디렉토리 확인
dir / ls(리눅스)
. -> 현재 디렉토리
.. -> 내 상위 디렉토리

디렉토리 간 이동
cd (change directory)

화면 지우기
cls

디렉토리 생성
mkdir

파일 복사
copy

파일 삭제 del

디렉토리 삭제 rmdir

명령어 옵션
/s 하위폴더
/q 묻지마라

윈도우에서 파워쉘창 여는법
해당 폴더에서 shift + 마우버튼

shell
명령어를 입력하는 곳
폴더에서 shift 우클릭

파워쉘 or 리눅스 창 복붙
붙혀넣기 : Shift + Insert
복사하기 : Ctrl + Insert
