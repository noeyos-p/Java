- Java 설정
찾기 -> 시스템 환경 변수 편집
환경 변수 Path 들어가서
Java 17에 있는 것 삭제 후
C:\Program Files\Java\jdk-17
새로 추가 하기 + 위로 올리기

Java 8 설치하고
두버전이 나오는데 jdk-8로 윗파일 수정
환경 변수 Path 들어가서
Java 8에 있는 것 삭제 후
환경 변수 JAVA_HOME 만든 후
C:\Program Files\Java\jdk-8
만듦

환경 변수 Path 들어가서
원래있던 자바 모두 삭제 후
%JAVA_HOME%\bin
새로 추가 하기 + 위로 올리기

환경 변수 JAVA_HOME 들어가서
Java 17 or 8로 jdk-17 or jdk-8로 변경하면
어떤 버전으로 변경했냐에 따라
cmd에 뜨는 자바 버전이 다름

cmd 종료 후 재시작해서
java -version
17버전으로 설정했다면
java version "17.0.12"
8버전으로 설정했다면
java version "8.0.12"
라고 떠야함
-> Java 세팅 완료
